Welcome To Freshman

I hope this website will help you figure out the tools you might use when you begin your College IT journey. This website in no means is a tutorial in getting started in IT, however I have found that beginning with a proper set of tools can provide a better start.

When I began to learn programming, I was unaware of the great tools and softwares that could have helped me to write better codes. I began my programming journey when I was in Grade 6, before that I didn't know that I could write codes to instruct my computer to do stuff for me. The first programming language that I learned was QBASIC. QBASIC provided this interface where you could write code in a old looking window and press F5 to execute the code. I didn't knew it back then but what I used was called an IDE(Integrated Development Environment), a software where you could write and execute your code in the same place. However this prevented me from learning another language, because when I tried to learn Python I couldn't find the QBASIC like IDE I was used to, I used to believe that you could program in only such software, things got worse when I was writing HTML in Window's Notepad.

I hope you can use the information in this website and speedup your learning process.

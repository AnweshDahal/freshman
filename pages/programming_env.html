Introduction

Programming environment is a set of softwares which you will use while writing programs & creating softwares. Programming environment usually contain a Compiler or an Interpeter, Editors, & Debuggers. A beginner programmer usually needs to be fimiliar with a Compiler/ Interpeter, and Editors. You will use Debuggers once you start writing huge programs, for now you might want to debug your code on your own through analysis.

Programming Language

You will use syntatically specific instruction (Language) while writing programs, these languages are called Programming Language. There are many types (classifications) of programming language which you might use in your programming career as per your requirement. Some examples are:

  1. Compiled Programming Language
  2. Interpreted Programming Language
  3. Procedural Programming Language
  4. Functional Programming Language
  5. Object Oriented Programming Language

But, for now we will look at the two main classifications of programming language, Compiled Programming Language & Interpreted Programming Language . The code you write is written in english language, but your computer can't understand instruction written in english so we use a translater like program to convert our english code to whichever ( Most probably Binary Code) language your computer speaks.

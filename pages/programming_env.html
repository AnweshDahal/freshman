Introduction

Programming environment is a set of software which you will use while writing programs & creating softwares. Programming environment usually contain a Compiler or an Interpeter, Editors, & Debuggers. A beginner programmer usually needs to be fimiliar with a Compiler/ Interpeter, and Editors. You will use Debuggers once you start writing bigger programs, for now you might want to debug your code on your own analysis.

Programming Language

You will use syntatically specific instruction (Language) while programming, these languages are called Programming Language. There are differnt types (classifications) of programming language which you might use in your programming career as per your requirement. Some examples are:

  1. Compiled Programming Language
  2. Interpreted Programming Language
  3. Procedural Programming Language
  4. Functional Programming Language
  5. Object Oriented Programming Language

But, for now we will look at the two main classifications of programming language, Compiled Programming Language & Interpreted Programming Language . The code you write is written in english language, but your computer can't understand instruction written in english so we use a translater to convert our english code to whichever ( Most probably Binary Code) language your computer speaks.

Choosing Your First Programming Language

Choosing your first programming language is very importent, because the impact it makes on you will determine how far you will go on your IT career, Try choosing a language YOU find inetersting. Don't search the web for Best Programming Languages OR Top 10 Programming Languages and take the first suggestion. However I suggest you to choose at least 5 different programming language and try the following test.

Write a simple program that asks the user for their first name and favorite number, then generate their username in the following format.

[FirstNameFavouriteNumber]

For Example:
First Name = Dennis
Favorite Number = 42
Username = Dennis42

Then ask yourself, which programming language was the most fun to work with. Finally, you can continue with the program you like.

There a lots of online tools you could use to try out a programming language without installing it to your system. My favorite is JDoodle.

Setting Up Some Programming Languages

I will assume you know how to install programs using installers in Windows
Lets setup some programming languages, lets assume you decided to use Python . We can start by downloading the Python Interpeter.

Make sure, that you download Python 3.x.x
Go to Python 3.x Download Page

You can select all the default options, however make sure that you select the checkbox that says the following.



However, if you forgot to select the option you can always re-execute the Python installer and select the Modify button.


Proceed throught the installation and select the option. After installation open up a terminal, I will use the Git Bash.


Now run the following command
python -V
The version of the Python Installation must be printed on the screen.

Congratulations, you installed Python. Next we will install Java as an example of Compiled language. To setup Java™ you had to install 2 peices of software JRE(Java Runtime Environment), it was used to execute codes and programs written in Java, next you need JDK (Java Development Kit), it was used to develop (Compile) Java code. However, you can install the latest version of Java SE (Java Standard Edition), which allows you to develop and run your Java code.

Go to Java SE 14 Download Page
Make Sure that you Install the Latest Java SE

After you download the JavaSE, run the installer. You can select the default options during installation. After you've finished the installation you have to set the environment variables.

  • Go to the Java installation directory, this is usually in [C:/Program Files/Java/jdk-[your version]]
  • Inside that directory there will be a folder called bin copy the path of that directory You can copy the path of a folder by Right clicking the folder while holding Shift and selecting the "Copy as path" option
  • Search for Environment Variable on your Windows™ search bar
  • Select the option that says Edit the system environment variables
  • Select the button that says Environment Variables...
  • With the Path variable selected click on the Edit button.
  • The Edit environment variables window opens, now you can click on the New button and paste the path we copied earlier.
  • Before Exiting, make sure that no other Java entry exists, if it does just move them below the entry we just added. This makes sure that the compiler and the executor both use the same Java version.
  • To check your installation you can go to the terminal and enter the following command.
    java -version
    And
    javac -version
    The outputs must point to the same Java version.
